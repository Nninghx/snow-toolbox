## 音频提取工具
### 使用方法
1. 双击运行 `Audio tool\Yin Pin Ti Qu V2-2-1.py`
2. 点击"浏览..."按钮，选择要提取音频的视频文件（支持 .mp4、.avi、.mkv、.mov 格式）
3. 设置输出音频文件的保存路径和文件名（默认为MP3格式）
4. 点击"开始提取"按钮
5. 等待提取完成，程序会提示成功或失败
### 界面说明
| 控件 | 说明 |
|------|------|
| 选择视频文件 | 输入或浏览选择源视频文件路径 |
| 输出音频文件 | 设置提取后音频的保存位置和文件名 |
| 浏览... | 打开文件选择对话框 |
| 开始提取 | 执行音频提取操作 |
| 帮助 | 查看更多帮助信息 |
